杭州市城市规划档案馆简介

 

　　二○○三年五月二十三日经杭州市机构编制委员会批准,同意杭州市城市规划信息中心增挂杭州市城市规划档案馆牌子。杭州市城市规划档案馆的主要工作职责是：贯彻执行档案工作的法律、法规和方针政策,负责全市城市规划审批档案的收集、管理、开发利用、咨询服务和信息化工作。
　　杭州市城市规划信息中心前身为杭州市城市规划信息所,成立于一九九二年,隶属于市规划局。二000年四月六日经杭州市编委杭编办[2000]22号文批准,更名为杭州市城市规划信息中心。中心内设四个科室,即办公室、信息科、综合档案科、技术服务科。
　　自一九九二年成立以来,信息中心综合档案主要分为二大块,一块是中心本身职能活动中形成的具有保存价值的文书材料和资料。另一块是局规划审批业务中形成的建筑、征地等文件、图纸和资料。信息中心的文书档案共128卷,其中永久58卷,长期48卷,短期22卷,会计档案200卷:局规划审批档案案卷总数略。　　
　　按照市编委批准的文件,杭州市城市规划信息中心统一负责规划审批档案及档案资料的信息化处理,因此中心除了要管理好本单位的各类档案以外,还要承担局综合档案的管理,同时还要检查、指导全市规划审批档案。尤其是规划审批档案中包括建设项目报建审批和查处违法建设两项主要内容,因此档案内容十分复杂。一方面档案数量较大。中心本身的档案数量虽不多,但局规划管理业务档案,每个建设项目的报建审批从开始到结束有一套完整的文件、图纸和资料,每个违法建设项目的立案查处也有一套完整的案卷材料同时随着规划管理职能的调整、市区行政区划调整后城市规划区范围的扩大,近年来各类文件、资料、材料的数量明显增加。另一方面档案利用率高。不仅要承担局机关内部日常工作中需经常性查阅参考,而且在对外提供查考、咨询服务工作中也发挥了越来越重要的作用,有时还成为行政执法和司法机关的办案证据。杭州市规划局是市政府主管城市规划工作的一个重要职能部门,规划基础建设的规划档案工作是市规划局城市规划的重要组成部分,在我局工作中具有举足轻重的重要地位,并将日益发挥着越来越重要的作用。
